# KaZeTab
KaZeTab is a Chrome extension that turns the new tab page into a calm, customizable wallpaper experience with quick search and bookmarks.

## Features
- Choose a wallpaper style like `coding`, `nature`, `anime`, or `space`
- Type your own custom genre and fetch matching wallpapers
- Use one wallpaper style all day or assign different moods by time of day
- Refresh the current wallpaper from the popup
- Show or hide bookmarks on the new tab page
- Optional time-based greetings
- Search directly from the new tab page
- Clean settings page for customization

## How to use
- Open Chrome and go to `chrome://extensions/`.
- Turn **Developer mode** on.
- Click **Load unpacked**.
- Select this project folder.
- Open the extension popup from the toolbar.
- Use **Settings** to choose your wallpaper mode and preferences.
- Open a new tab to test the saved experience.

## Notes
- The extension fetches wallpapers from Wallhaven.
- If you change files locally, reload the extension from `chrome://extensions/` before testing again.
